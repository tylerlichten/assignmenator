# Assignmenator

Assignmenator swaps "a" in a string for an "X".

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'assignmenator'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install assignmenator

## Usage

Use method "marks_the_spot" for swapping string chracters.
